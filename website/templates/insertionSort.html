Insertion Sort

Insertion Sort is a straightforward in-place sorting algorithm that builds the sorted array by gradually inserting each element into its correct position. It works well for small datasets or nearly sorted data.

1. Initialization

  • Assume the first element is sorted (considered as a sorted subarray).

2. Iterative Comparison

  • For each unsorted element, compare it with elements in the sorted subarray, moving elements greater than it to the right.

3. Insertion

  • Insert the current element into its correct position in the sorted subarray.

4. Repeat

  • Repeat these steps until the entire array is sorted.

About Insertion Sort

Insertion Sort is an in-place algorithm, meaning it doesn't require additional memory. While it is not the most efficient for large datasets, it performs well on small datasets or partially sorted data. The algorithm has a time complexity of O(n^2) in the worst case but can be efficient for small or nearly sorted arrays.

    • Speed

    Time complexity

    • Insertion Sort's average-case time complexity is also O(n^2), making it less efficient than more advanced sorting algorithms like Merge Sort or QuickSort for large datasets. Despite its higher time complexity, Insertion Sort has its merits for small datasets or partially sorted data due to its simplicity and low overhead.

    Space Complexity (Memory)

    • The space complexity of the Insertion Sort algorithm is O(1), indicating that it uses a constant amount of extra space regardless of the input size. Insertion Sort is an in-place sorting algorithm, meaning it doesn't require additional memory proportional to the size of the input.
